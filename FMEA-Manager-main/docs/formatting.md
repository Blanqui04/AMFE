AMFE Workbook Formatting Overview
---------------------------------
1. **Title Row (Row 1)**  
   • Merged A1:AE1  
   • Arial, Bold 14‑pt, centered  
   • Border all around, fill with professional blue tone (B7DEE8)

2. **Stage Headers (Row 2)**  
   • Defined stage blocks (e.g. A‑B, C‑E, etc.), each merged  
   • Arial, Bold 10‑pt, centered  
   • Unique fills: blue, green, peach, yellow, lavender, blue  
   • Thin-black cell borders throughout

3. **Column Titles (Row 3)**  
   • Specific columns filled by category:  
     – Light yellow, grey, blue, purple for groups  
     – Green (#A9D08E) for severity/occurrence/detection/action priorities (some rotated 90°)  
     – Autofit column widths  

4. **Global Formatting (All other cells)**  
   • Arial 10‑pt, centered both horizontally/vertically  
   • Thin-black cell borders  
   • Freeze top 3 rows for easy navigation

5. **Sizing**  
   • Columns width: 10–25 pt based on content length  
   • Row 1 height: 25 pt; rows 2–3 height: 20 pt

This ensures a clean, consistent, and professional format tailored for AMFE process analysis.
